## 🔒 POLÍTICA DE PRIVACIDAD Y SEGURIDAD

### Compromisos de RM:
1. **Transparencia Total:** Todos los datos se procesan localmente, sin envíos externos sin consentimiento
2. **Seguridad Máxima:** 
   - Tokens encriptados
   - SYS_PTRACE habilitado para debugging seguro
   - Seccomp deshabilitado solo en ambiente controlado
3. **Sin Vigilancia:** No se rastrean usuarios ni se recopilan datos biométricos
4. **Acceso Controlado:** Solo usuarios autenticados
5. **Auditoría:** Todos los cambios se registran en Git
